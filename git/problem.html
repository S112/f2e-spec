# 1. Failed to connect to github.com port 443: Operation timed out

解决方法：

  • 查找github.com (opens new window)
  • 查找github.global.ssl.fastly.net (opens new window)
  • 查找assets-cdn.github (opens new window)
  • 添加到本地host文件中

199.232.69.194 github.global.ssl.fastly.net
