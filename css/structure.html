    # 目录结构

    ├── style
│   |── base.scss
│   |── common.scss
|   |── element.scss (名称可变)
│   |── func.scss     
│   |── reset.css
│   |── utils.scss     
│   |── vars.scss

    # (1) base.scss

    style目录的唯一开放入口，供 main.js 引入。

    # (2) common.scss

    从业务组件中，提取的公共样式。

    # (3) element.scss

    自定义的第三方UI组件库样式，如element、ant、vant、iview等。

    # (4) func.scss

    scss混合(mixin)

    # (5) reset.css

    样式重置文件

    # (6) utils.scss

    工具类样式：清浮动、文字省略、1px边框等。

    # (7) vars.scss

    自定义变量，如颜色色值、文字大小等。
