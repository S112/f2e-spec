    # 1. 父子组件生命周期执行顺序

    • 加载渲染过程
      父： beforeCreate -> created -> beforeMount
      子： beforeCreate -> created -> beforeMount -> mounted
      父： mounted
    • 更新过程
      父： beforeUpdate
      子： beforeUpdate -> updated
      父： updated
    • 销毁过程
      父： beforeDestroy
      子： beforeDestroy -> destroyed
      父： destoryed
